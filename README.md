# 🛠️ Useful Daily Scripts

A collection of scripts designed to streamline and simplify everyday development tasks.

## ⚙️ Odoo-cli
A command-line tool to easily manage Odoo servers, configurations, and operations — making Odoo development faster and more efficient.

👉 For setup instructions, refer to the [odoo-cli](odoo-cli/README.md) guide.

## 🔀 conflict-detector
A script to help detect and analyze Git merge conflicts across branches or commits.  
Use it to quickly spot conflicting areas and improve merge workflows.

## 💻 vscode

Project support files and configuration for Visual Studio Code.  
Includes recommended settings, workspace configurations, and helpful .vscode templates to boost productivity when working on Odoo development.

👉 See the [vscode](./vscode) directory for defaults and guidance.
